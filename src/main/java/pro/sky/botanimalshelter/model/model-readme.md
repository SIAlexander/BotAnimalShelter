Entities for Pet Shelter Telegram Bot will be created in this package

В этой директории будут созданы сущности для Телеграм-бота приюта домашних питомцев

Сущности

**Питомцы**

Собака
Кошка

Ветеринар
Специалист по внешнему виду питомцев
